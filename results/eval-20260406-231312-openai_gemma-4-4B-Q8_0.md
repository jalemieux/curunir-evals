### ⚙️ Simple Function Call (e.g., `read`, `web-search`, `grep`)

**What it is:** A direct, atomic request to execute a single, well-defined action using a specific tool. The entire process—input, execution, and output—occurs within the current conversational context.

**When to Use It:**
*   **Atomic Tasks:** When the required operation is a single, discrete action (e.g., "Fetch the content of this URL," "Count lines in this file," "Search for a specific string").
*   **Immediate Feedback:** When you need a quick result that can be integrated directly into the ongoing conversation flow.

### 🧠 `delegate` Tool

*   **High Complexity / Multi-Step Reasoning:** When the task requires a series of logical steps, analysis, synthesis, or iteration (e.g., "Research three competitors, compare their pricing models, and draft a SWOT analysis").
